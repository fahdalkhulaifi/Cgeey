Lets talk about Keyboard shortcuts

If you are checking out this blog, then most probably you are a developer.

Today, I will show you keyboard shortcuts, which will sharpen up your keyboard skills.

The key to learning is-

Practice! Practice! Practice!

You will reach a point where keyboard shortcuts will come at ease to you and you will start using them naturally.

Now enough of theory!! Lets jump to the practical part.

0. Home key

Moves your cursor to the start of the line

This is one of my fav keys :) Just magical

1. End key

Moves your cursor to the end of the line

Another magical key.

2. Shift + Home

Selects from current cursor position to start of line

3. Shift + End

Selects from current cursor position to end of line

4. Ctrl + Home

Moves cursor to start of the current document

5. Ctrl + End

Moves cursor to end of the current document

6. Ctrl + Right arrow

Moves cursor to next word or end of current word

7. Ctrl + Left arrow

Moves cursor to start of current word

8. Next Error (F2)

Moves cursor to next error

9. Previous error (Shift + F2)

Moves cursor to previous error

10. Duplicate line (Ctrl + D)

11. Expand Selection (Ctrl + W)

12. Go to File (Ctrl + N)

13. Go to Class (Ctrl + Shift + N)

14. Add Dart File (Alt + A, Alt + D)

Change it according to your preference, maybe Kotlin or Java for Android

15. Add Package (Alt + A, Alt + P)

16. Refactoring (Shift + F6)

17. Go to declaration (Ctrl + B)

18. Show Quick actions (Ctrl + Shift + A)

Use it when you forget any shortcut, you will find most of them here

19. Move Cursor to next hump (Alt + Right arrow)

20. Select till next hump (Shift + Alt + Right Arrow)

21. Move Cursor to previous hump (Alt + Left arrow)

22. Select till previous hump (Shift + Alt + Right Arrow)

23. Delete till previous hump

This is how you should feel right now-

Thank you for staying till the end
